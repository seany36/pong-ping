# pong-ping

Ping Pong Game:
This repository contains a simple implementation of a classic ping pong game using Python and Pygame.

Features:
Player vs Player mode
Score tracking
Sound effects

Controls:
Player 1: W (Up) and S (Down)
Player 2: Arrow keys (Up and Down)
